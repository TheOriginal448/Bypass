Webretro

Webretro is an online emulator for retro games.

Download Standalone

Using Webretro:

Use my ROMS page to get the ROMs of the games that you want. While on the main screen of Webretro, select AutoDetect and upload the .zip file of the ROM file for the game. Select Controls in the top left, then select Keybinds. Look at what each button on the game controller pairs to on your keyboard.

Saving:

Click the saves button, then click save game.
