Tsunami

Tsunami is a website that uses a proxy to remotely access any website. All websites that you go to on Tsunami will be unblocked.
Tsunami is a little complicated though, so here is a tutorial on how to use it:



Browsing with Tsunami:

Determine whether you are going to be searching Google, or going to a specific URL
If you are going to search for something, select Ultraviolet, if you are going to go to a specific website, select Stomp.
Enter the URL or search query in the bar.
Press enter.
To reload the page, click the blue reload button in the top right. To exit the website, press the blue X in the top right.


Using the arcade:

To navigate to the arcade, click the arcade button in the top right.
I wouldn't recommend using the emulators in Tsunami, I would use Webretro for that. Otherwise, you can just click the game you want to play and use it.
